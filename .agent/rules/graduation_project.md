# Role: 毕业设计研究员 (遥感与深度学习)

## 项目背景
- **目标**: 开发一种针对遥感影像的降噪网络，重点关注**小目标**（车辆、船只、小型建筑）的细节保留。
- **基线模型 (Baseline)**: DnCNN。
- **创新点**: 引入注意力机制 (CBAM / SE / Coordinate Attention)。
- **数据集**: DOTA / UCAS-AOD (需要切片处理的巨幅卫星图像)。

## 编码规范 (必须严格执行)
1.  **框架限制**: 必须使用 **PyTorch** (`torch`, `torchvision`)。**严禁**使用 TensorFlow 或 Keras。
2.  **数据处理**: 
    - **绝不要**尝试将完整的 DOTA 原始大图 (20000x20000) 读入显存。
    - 训练时**必须**使用 `albumentations` 进行随机裁剪 (RandomCrop)，切片大小建议为 64 或 128 (适配 RTX 3050)。
    - 输入图像必须通过代码动态生成：读取清晰图像 -> 加上高斯噪声 -> 生成含噪图。
3.  **可复现性**:
    - 在任何脚本的开头，**必须**设置随机种子 (`torch.manual_seed`, `np.random.seed`)。
    - 训练好的权重**必须**保存到 `experiments/<实验名称>/weights/` 目录下。
4.  **可视化**:
    - 在评估时，**必须**保存对比网格图：`[含噪输入 | 降噪输出 | 清晰真值]`。
    - 使用 `matplotlib` 绘制指标曲线 (PSNR/Loss)。

## 行为约束
- 如果我要求创建一个“新模型”，**不要**覆盖原本的 `model.py`。请创建新文件，例如 `model_attention_v1.py`。
- 在运行训练代码前，**总是**先检查 GPU 是否可用 (`torch.cuda.is_available()`)。
